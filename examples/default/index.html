Best Practices

  • Guideline for most of elements, which are used on most of pages.
  • Global
    • Please make sure that the page is valid HTML!
    • Use `lang` attribute for the language of the page.
    • Use different `title` attribute for each page.
    • Use `alt` attribute for all images.
    • Use `title`, attributes for all elements which have a meaning and interaction.
    • Don't forget that users are using mouse or keyboard to navigate (specialy tab key).
    • Every user interaction has a visual feedback.
  • Form
    • Use labels for all inputs.
    • Use fieldset and legend for groups of inputs.
    • Use placeholder for examples of input values.
    • Checkbox, radio are still button. Use them with label.
    • Use select for dropdowns.
    • Every submit form has status message.
